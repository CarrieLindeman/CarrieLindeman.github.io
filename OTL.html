Faculty Resource Website

Faculty Resource Website
For the Office of Transformational Learning

Researcher & Designer, with support from Nikk Bonneau, Moises Melano, Doireann Renzi • January 2020 - November 2021

Key Skills: Information Architecture • User Interviews • Figma • Contextual Inquiry • Drupal Website Editing

Description

This project happened in three phases:

Research

  • Survey distributed to >100 individuals
  • 1:1 interviews with 15 people
  • Creating empathy maps
  • Heuristic evaluation of existing streaming platforms

Design

  • Vision boarding for desired aesthetic
  • Creating style guide and color scheme
  • Iterating on logo design

Mock-Ups

  • Using Figma to demo initial interactions
  • Using HTML, CSS, and JSON to create an interactive mock-up
  • Connecting the mock-up with a database of film & TV

Key features include a comprehensive filtering menu, a feature to sync preferences with a partner/roommate/etc, and taking in-the-moment quizzes to get detailed suggestions based on your current mood.
