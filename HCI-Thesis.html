Design for Just-in-Time

Resource Design for Self-Teaching Computer Science and Online Learning
HCI Master's Thesis

Researcher, Designer, & Author • June 2019 - May 2020

Key Skills: Interviews • User Tests • Research Coding • HTML Prototyping

Users: Adult novices who are self-teaching coding

How might we:

  • investigate how just-in-time resources may support self-teaching for new coders
  • explore how the design of just-in-time resources effect users
  • consier design changes for popular existing resources



Project Phases

Phase 1

  • Observation of user with existing resources
  • Determine frustrations and rewards
  • Coding findings from observations into themes

Phase 2

  • Identifying design choices to meet needs
  • Creating design sketches
  • Implementing Design Mock-up

Phase 3

  • Observation of user with re-designed resource
  • Holding post-observation interviews
  • Coding interviews into themes
